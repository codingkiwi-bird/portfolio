story-GPT

스토리 생성 인터랙션을 위한 GPT 기반 웹 경험

사용자 참여형 시나리오를 구성하고 대화형 진행을 지원하는 프로젝트

JavaScript HTML/CSS LLM Prompt

프로젝트 개요

React를 활용한 프론트엔드와 Node.js 기반의 백엔드로 구성된 풀스택 전자상거래 플랫폼입니다. 사용자가 쉽게 상품을 검색하고 구매할 수 있는 직관적인 인터페이스를 제공하며, 관리자는 효율적으로 상품과 주문을 관리할 수 있습니다.

주요 기능

  • 사용자 회원가입 및 로그인 시스템
  • 상품 검색 및 필터링 기능
  • 장바구니 및 위시리스트 관리
  • 안전한 결제 시스템 연동
  • 주문 내역 및 배송 추적
  • 관리자 대시보드 (상품/주문 관리)

기술적 구현

Frontend

React, Redux, React Router, Styled Components

Backend

Node.js, Express.js, JWT Authentication

Database

MongoDB, Mongoose ODM

기타

Stripe API, Cloudinary, Heroku

개발 과정에서 배운 점

이 프로젝트를 통해 풀스택 개발의 전체적인 흐름을 경험할 수 있었습니다. 특히 사용자 인증, 결제 시스템 연동, 데이터베이스 설계 등 실제 서비스에 필요한 핵심 기능들을 구현하면서 실무에 가까운 개발 경험을 쌓을 수 있었습니다.
