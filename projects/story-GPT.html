OpenAI API를 활용한 스토리텔링 웹 서비스

사용자가 선택지를 고르며 이야기를 전개해 나가는 인터랙티브 스토리 플랫폼입니다. OpenAI API를 통해 장면마다 자연스러운 내러티브를 생성하고, 선택 결과에 맞춘 다음 전개를 즉시 이어갑니다.

HTML CSS JavaScript Node.js SQLite OpenAI

프로젝트 목표

OpenAI API를 이용해 사용자의 프롬프트와 현재까지의 플롯을 바탕으로 다음 장면의 스토리를 생성합니다. 생성되는 스토리 및 선택지들이 정상적인 범주내에서 출력되도록 프롬프트 엔지니어링을 진행합니다. 최종 스토리를 출력후에는 저장하고 공유할수 있는 플렛폼을 제공합니다.

주요 기능

  • 스토리 생성
  • 선택지 생성 및 선택
  • 선택지에 따른 새로운 스토리 생성
  • 완성된 스토리 공유
  • 공유된 스토리 감상

기술적 구현

Frontend

HTML, CSS, JavaScript

Backend

Node.js (Express)

Database

SQLite

etc

개발 과정에서 배운 점

초기에는 단일 스토리 생성 API로 모든 단계를 처리해 인트로와 엔딩이 이상하게 출력되는 문제가 반복되었습니다. 엔드포인트를 인트로·전개·엔딩으로 분리하고 단계별 프롬프트와 출력 제약을 달리 적용하여 프롬프트 엔지니어링을 통해 인트로는 시작하는 느낌과 엔딩은 끝나는 느낌으로 결과가 안정적으로 생성되어 전반적인 스토리 완성도가 개선할 수 있었습니다.
