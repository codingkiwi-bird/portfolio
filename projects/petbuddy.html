팀 협업과 업무 효율성을 향상시키는 프로젝트 관리 플랫폼

Vue.js Firebase SCSS PWA

프로젝트 개요

Vue.js와 Firebase를 활용하여 개발한 실시간 프로젝트 관리 애플리케이션입니다. 팀원들이 실시간으로 협업할 수 있는 환경을 제공하며, 칸반 보드 형태의 직관적인 인터페이스로 프로젝트 진행 상황을 한눈에 파악할 수 있습니다.

주요 기능

  • 실시간 칸반 보드 (To Do, In Progress, Done)
  • 팀원 초대 및 권한 관리
  • 태스크 생성, 수정, 삭제
  • 댓글 및 파일 첨부 기능
  • 프로젝트 진행률 대시보드
  • 알림 및 데드라인 관리
  • PWA 지원으로 모바일 앱처럼 사용 가능

기술적 구현

Frontend

Vue.js 3, Vuex, Vue Router, SCSS

Backend

Firebase Firestore, Firebase Auth

기타

PWA, Service Worker, Firebase Hosting

개발 과정에서 배운 점

Firebase의 실시간 데이터베이스를 활용하여 실시간 협업 기능을 구현하는 과정에서 NoSQL 데이터베이스의 특성과 실시간 데이터 동기화에 대해 깊이 있게 학습할 수 있었습니다. 또한 PWA 기술을 적용하여 웹앱의 사용자 경험을 네이티브 앱 수준으로 향상시키는 방법을 익혔습니다.
